# 个人博客

这是我的个人技术学习博客。

我做这个博客，主要是想把自己在学习和工作里接触到的内容整理下来。现在我的方向主要还是 Python 爬虫、数据采集、JS 逆向、APP 抓包分析这些，所以博客里写的内容也基本围绕这些展开。

对我来说，这个博客不是单纯拿来展示的页面，更像一个长期积累的学习记录。很多时候一个问题当下解决了，如果不记下来，过一段时间还是会重新踩一遍。所以我想把平时遇到的问题、排查过程、一些项目里的思路，还有学到的新工具和新方法都慢慢整理进去。

目前这个博客里会主要记录这些内容：

- Python 爬虫开发和项目实践
- Scrapy、Requests、Selenium、Playwright 的使用过程
- JS 逆向、接口排查、Cookie / Token / 签名问题分析
- Charles、Wireshark 等抓包工具相关笔记
- 数据清洗、分类、去重、入库这些采集后处理内容

后面我也会继续补更多自己的学习笔记，让这个博客更像一个持续更新的个人技术记录，而不只是一个静态主页。

## 本地预览

直接打开 `index.html` 就可以预览。

如果想用本地服务方式打开，可以在当前目录执行：

```powershell
python -m http.server 8080
```

然后访问：

```text
http://127.0.0.1:8080
```
