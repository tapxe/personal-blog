# crawler-blog

基于你的简历内容生成的个人博客静态站，适合展示以下方向：

- 爬虫工程
- JS / App 逆向
- 数据采集工程化
- 个人履历与联系方式

## 使用方式

直接双击打开 `index.html` 即可预览。

如果你想用本地服务打开，也可以在当前目录执行：

```powershell
python -m http.server 8080
```

然后访问 `http://127.0.0.1:8080`。

## 可继续扩展

- 把博客文章拆成单独页面
- 接入评论系统
- 增加文章后台或 Markdown 生成流程
- 绑定你自己的域名并部署到 Vercel / Netlify / GitHub Pages
